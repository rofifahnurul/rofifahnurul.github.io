Context Free Grammars

Context Free Grammars adalah adalah cara mendeskripsikan bahasa dengan aturan rekursif yang disebut produksi. CFG terdiri dari sekumpulan variabel, a set Setiap string nol atau simbol terminal, dan variabel start, serta hasilnya. produksi terdiri dari variabel kepala dan b o dy terdiri dari a lebih banyak variabel dan / atau terminal.

Apa itu Context Free Grammars (CFG) ?

Manusia saat ini menginginkan kemudahan. Salah satu bentuk kemudahannya yaitu untuk melakukan pencarian saat ini tidak lagi melakukan pencarian secara manual, sebagai gantinya mesin bekerja untuk mencapai apa yang diinginkan manusia. Mesin pencari berkembang di sistem operasi, internet, bahkan editor teks. Mesin melakukan pencarian dengan mencocokkan pola yang diberikan dengan tabel indeks (teks) yang tersedia. Dengan regular expression, mesin dapat melakukan pencarian ini. Regular Expression (Regex) merupakan sebuah teks (string) yang mendefinisikan sebuah pola pencarian sehingga dapat membantu kita untuk melakukan matching (pencocokan), locate (pencarian), dan manipulasi teks. Dalam regex ada beberapa metakarakter untuk menyatakan bermacam-macam teks, contohnya yaitu ^ merupakan tanda awal string dan $ merupakan tanda akhir string.

Context Free Grammars dalam Parsing

Parsing adalah cara mendeskripsikan struktur kalimat. Ini merupakan hal yang penting ketika ita perlu memahami arti sebuah kalimat, seperti yang kita lakukan misalnya dalam menerjemahkan dari satu bahasa ke bahasa lain. Dalam ilmu komputer, ini relevan dalam interpreter, compiler, dan program penerjemahan lainnya.

Grammar
Penjelasan

Dari grammar diatas dapat dilihat ada beberapa aturan produksi. S -> &infinite; maksudnya nanti kita bisa mengubah Terminal menjadi sebuah string kosong. Lalu untuk i menggambarkan if dan e menggambarkan else. Contoh parsing :
S -> SS
S -> iSS
S -> iiSeS
S -> iie

Jadi dari grammar tersebut salah satunya kita bisa mendapatkan string iie.

Dengan grammar diatas kita juga bisa mengecek apakah suatu string berasal dari grammar tersebut. Misalnya untuk string iieee. i pertama cocok dengan e pertama (dari sebelah kiri). Maka ini dihapus. Kemudian i kedua cocok dengan e kedua (dari sebelah kiri). Maka ini juga dihapus. Tetapi masih tersisa 1 e, sedangkan i sudah tidak ada. Sehingga string iieee ini bukan merupakan language karena tidak sesuai dengan polanya.

Programming
Penjelasan

Hubungan dari pola grammar tersebut sesuai dengan pola if else pada C program. Dimana tidak boleh ada else yang lebih banyak dibanding if. Maksudnya if bisa muncul dengan else atau tidak, tetapi else hanya bisa ada jika ada if.

Kita telah melakukan parsing dan mendapatkan string iie yang valid dari grammar yang ada di contoh. Lalu, gambar diatas adalah implementasi string iie dalam program C. Dari pencocokan string ini dapat membantu kita dan compiler untuk mengetahui pasangan if dan else yang cocok.
